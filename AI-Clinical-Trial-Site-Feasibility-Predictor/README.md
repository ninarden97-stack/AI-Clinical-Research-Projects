# AI Clinical Trial Site Feasibility Predictor

This prototype explores how AI could support clinical trial site selection by predicting the likelihood that a site will successfully recruit patients.

Selecting appropriate sites is a major determinant of trial success. Historical site performance data combined with machine learning models could help sponsors identify high-performing sites earlier in the planning process.

This project demonstrates a conceptual workflow for such a system.

## Example Use Case

Clinical trial teams must decide which investigator sites are most likely to recruit participants successfully.

This prototype demonstrates how a machine learning system could analyze historical site performance and protocol characteristics to predict recruitment feasibility.

Outputs include:

- feasibility scores for candidate sites
- predicted enrollment potential
- key operational risk drivers
- recommended mitigation actions

These insights could support clinical operations teams when selecting and prioritizing investigator sites for new studies.
